Terminal üzerinden proje klasörümüzün içine girip npm install komutunu çalıştırarak node_modules dosyalarını yüklüyoruz
Yine klasör içerisinde npm start komutunu çalıştırarak projemizi ayağa kaldırıyoruz
Karşimiza gelen a(Android) veya i(ios) seçeneklerinden birini seçip klavyeden a veya i girişi yapmamız yeterli. 
(Not: Eğer ki telefonunuza Expo uygulamasını kurarsanız npm start yaptıktan sonra size verilen kare kod ile cihazınızdan giriş yapabilirsiniz.)
Uygulama firebase realtime Database e balı durumdadır. Fakat src/firebaseConfig.ts içerisinden kendi firebase hesabınızın ayarlarını girerek kendi firebase hesabııza da bağlayabilirsiniz
